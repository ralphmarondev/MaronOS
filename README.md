# 🪄 Maron OS

A virtual operating system experience inside an Android app.

---

## 🌟 Features

- Setup Wizard
- Auth Screen
- Default Local Account (`root` / `toor`)
- Firebase Sync
- Notes
- Calculator
- Browser
- Clock
- Music Player
- Gallery
- Camera
- Settings

---

## 🚀 Tech Stack

- Kotlin
- Jetpack Compose
- Material 3
- Koin
- DataStore
- Firebase Realtime Database
- CameraX
